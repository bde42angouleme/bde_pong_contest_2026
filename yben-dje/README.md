# Pong

## Description

A pong. How to describe better?

The pad's movements are smoothed using Nvidia's DLSS 5.

## Instructions
### Build

To build the project, you can run this command
```bash
make
```

Recompile the project with
```bash
make re
```

### Execute

```bash
./pong
```
